# Proyecto .NET MAUI

Este proyecto es una aplicación .NET MAUI que tiene como objetivo proporcionar una experiencia de usuario multiplataforma utilizando .NET 9. 

## Requisitos

- .NET 9 SDK
- Visual Studio 2022 (o superior) con soporte para .NET MAUI

## Instalación

1. Clona el repositorio: https://github.com/agarciamears/taskManagerMaui.git
2. Navega al directorio del proyecto: `cd taskManagerMaui`
3. Restaura las dependencias del proyecto: `dotnet restore`

## Estructura del Proyecto

- **/Platforms**: Contiene el código específico de cada plataforma (Android, iOS, Windows, etc.).
- **/Resources**: Contiene recursos como imágenes, fuentes y archivos de diseño.
- **/Views**: Contiene las vistas de la aplicación.
- **/ViewModels**: Contiene los modelos de vista que manejan la lógica de la interfaz de usuario.
